# News App

This is frontend challenge from innoscripta

### Steps to run the Application

1. Clone the repository:

   ```bash
   git clone https://github.com/Maryambashir/news_app.git

2. Install dependencies
    ```bash
    npm install

3. Run the application
    ```bash
    npm start

### Steps to run the Application with docker
1. Clone the project and Navigate to project directory
2. `docker build -t news_app .`
3. `docker run -it -p 3000:3000 news_app`
